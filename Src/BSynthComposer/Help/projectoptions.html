Project Options

The Project Options form is used to set options for the synthesizer that apply to all projects. The values in this form are stored in the registry and reloaded when BasicSynth starts.

The Composer field sets the default composer name. The Copyright field sets the default copyright notice. These values are used to initialize a new project.

The Project Tree Options buttons select optional features of BasicSynth that will appear in the project tree view.

  • Notelists shows the notelist file entry in the tree.
  • Sequences shows the sequencer file entry in the tree.
  • Script files shows an entry for scipt files. Script files are short programs that are processed by the synthesizer. (These are not yet implemented.)
  • Text files shows the text file entry in the tree. Text files are used to asscociate generic text files with the project. These can be used to hold program notes, etc.
  • Libraries shows the Library file entry in the tree. Libraries are collections of instruments.
  • SoundFonts shows the Sound Bank file entry in the tree. Enable this if you want to use sound bank instruments.
  • MIDI shows the MIDI entry in the tree. This is used to set default program and volume levels for MIDI channels. It is only needed if you have a MIDI keyboard.

The Projects field contains the default directory for project files.

The Wave-In field contains the default directory for WAV files used by the WFSynth insrument.

The Forms field contains the default directory for instrument form files.

The Colors field contains the file containing color definitions for instrument forms.

The Libraries field contains the default directory for instrument library files.

The MIDI In field allows selection of the MIDI input device.

The Wave Out field allows selection of the wave output device (sound card).

The Play Latency field contains the value in seconds for playback latency. A longer latency time creates a larger buffer for sound output. A larger buffer helps to prevent glitches in sound output, but produces a longer delay between sound generation and sound output. A value of 1 to 0.1 seconds is typical. However, when using a MIDI keyboard, this value should be set to the minimum amount (20 milliseconds) to avoid audible delays between key press and sound output.
